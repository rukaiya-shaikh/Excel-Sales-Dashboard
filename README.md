# Excel Sales Performance Dashboard

## Objective
To analyze sales performance using Excel and build an interactive dashboard.

## Dataset
Superstore sales dataset with 5000+ rows and 15 columns.

## KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Total Quantity

## Tools Used
- Microsoft Excel (Web)
- Pivot Tables
- Charts
- Slicers

## Key Insights
- Technology category has the highest sales.
- West region contributes the most revenue.
- Profit margin varies across categories.
